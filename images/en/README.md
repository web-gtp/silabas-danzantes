# Stimulus Images - English

Place PNG images of English words here.

## Naming Convention
- Format: `{word}.png`
- Recommended size: 256x256 px
- Background: Transparent (PNG)
- Style: Child-friendly drawings, colorful, clean background

## Example
- `sun.png` → image of the sun
- `cat.png` → image of a cat
- `butterfly.png` → image of a butterfly
