# Imágenes de Estímulos - Español

Coloca aquí las imágenes PNG de las palabras en español.

## Nomenclatura
- Formato: `{palabra_sin_acentos}.png`
- Tamaño recomendado: 256x256 px
- Fondo: Transparente (PNG)
- Estilo: Dibujo infantil, colorido, fondo limpio

## Ejemplo
- `sol.png` → imagen del sol
- `gato.png` → imagen de un gato
- `mariposa.png` → imagen de una mariposa
